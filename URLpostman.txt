4. Microservicio Logística (a través del API Gateway - Puerto 8081)

4.1 Asignar Transporte (Despachos)
URL: POST http://localhost:8081/api/logistica/despachos
JSON Body:
{
  "inventarioId": 10,
  "cantidad": 20,
  "vehiculo": "Camioneta 4x4",
  "horario": "2024-06-15T10:30:00"
}

4.2 Listar Despachos
URL: GET http://localhost:8081/api/logistica/despachos

4.3 Actualizar Entrega de Despacho
URL: PUT http://localhost:8081/api/logistica/despachos/{id}/entrega (Reemplaza {id})

4.4 Registrar Ingreso a Centro de Acopio
URL: PUT http://localhost:8081/api/logistica/ingreso/{trackingId} (Reemplaza {trackingId})

4.5 Listar Inventario
URL: GET http://localhost:8081/api/logistica/inventario

4.6 Listar Centros de Acopio
URL: GET http://localhost:8081/api/logistica/centros-acopio

4.7 Listar Centros de Acopio por Región
URL: GET http://localhost:8081/api/logistica/centros-acopio/region/{region} (Reemplaza {region}, ej: Metropolitana)
